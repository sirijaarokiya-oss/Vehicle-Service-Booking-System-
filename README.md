# 🚗 Vehicle Service Management System

A full-stack web application developed using **Java, JSP, Servlets, JDBC, and Apache Derby** to simplify vehicle service management. The system provides separate modules for administrators and users, enabling efficient service booking, mechanic management, and appointment tracking.

---

## 📌 Features

### 👤 User Module
- User Registration & Login
- Book Vehicle Service
- View Service Status
- Manage Profile
- Secure Authentication

### 🔧 Admin Module
- Admin Login
- Dashboard
- Add, Update & Delete Mechanics
- Upload Mechanic Images
- Manage Service Requests
- Update Service Status
- View Registered Users

### 🛠 Mechanic Module
- Mechanic Login
- View Assigned Services
- Update Service Progress
- Manage Profile

## 🚀 How to Run

1. Clone the repository.
2. Open the project in **NetBeans**.
3. Start **Apache Derby** database.
4. Configure the JDBC connection.
5. Deploy the project on **Apache Tomcat**.
6. Run the project.
7. Open in your browser.
